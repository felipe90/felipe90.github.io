# Felipe Rojas - Professional Portfolio

[Live Website](https://felipe90.github.io/)

Welcome to the repository of my personal portfolio and digital resume. This project serves as a comprehensive showcase of my career as a **Senior Frontend Engineer**, highlighting my professional experience, featured projects, and my modern workflow structured around **AI Oriented Coding**.

## 🚀 Features

- **Modern & Premium Design**: Custom UI/UX emphasizing storytelling, micro-interactions, and a clean, agency-level aesthetic.
- **Bilingual (i18n)**: Fully translated into English and Spanish using a custom JavaScript translation dictionary.
- **AI Oriented Workflow**: Showcases my "Vibe Coding" paradigm. The repository is continuously monitored by custom AI agents (via Gemini CLI) that automatically log development progress in real-time.
- **Responsive Layout**: Designed to provide an optimal viewing experience across all devices.

## 🛠 Tech Stack

- **Core**: HTML5, CSS3, Vanilla JavaScript
- **Styling**: Custom CSS variables, Flexbox, CSS Grid, CSS Animations, FontAwesome Icons
- **Workflow / Automation**: Gemini CLI, Custom Bash Scripts (`vibe-watcher.sh`)

## 💻 Running Locally

To run this portfolio locally on your machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/felipe90/felipe90.github.io.git
   ```
2. Navigate to the directory:
   ```bash
   cd felipe90.github.io
   ```
3. Start a local server (e.g., using Python 3):
   ```bash
   python3 -m http.server 8080
   ```
4. Open your browser and visit `http://localhost:8080/`.
